Super Simple Spotlights

Clean looking spotlights with jQuery.

Download this project as a .zip file Download this project as a tar.gz file

Super Simple Spotlights allows you to bring focus to an element on your page with jQuery.

Adding a spotlight is as simple as writing:

$(".element").spotlight();

Click me for a spotlight!

If you want, you can add a title to be displayed when the spotlight appears:

$(".element").spotlight( {title_html: "This a title to be shown when the spotlight appears, hello world!"} );

Click me for a titled spotlight!

By default, Super Simple Spotlights uses Font Awesome for the close button but you can easily override it:

$(".element").spotlight( {close_html: "Close me!"} );

Click me for a custom close button!

You can also bind to the close event of the spotlight:

$(".element").spotlight( {onclose: function(){ alert("You just closed a spotlight.")} } );

Click me for a binded spotlight!
